Atomic Flood

This game is my first created with the Unity game engine. I reused many of the assets from an earlier project I made in high school in addition to new ones in order to expand upon its ideas in a full-fledged game engine. Mechanics include jumping, a jetpack, shooting, fully-voiced NPC dialogue, doors and switches, and movable blocks. All art, music, and dialogue assets are my own creations. Tools I used include Unity, Visual Code Studio, Pixilart, Pix2D, LMMS, ttstool.com, and Audacity.
